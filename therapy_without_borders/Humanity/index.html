Welcome to Humanity



Who We Are

Therapists without Borders is an international humanitarian initiative dedicated to providing life-saving aid and sustainable solutions to communities facing war, displacement, poverty, and climate disasters.


Our Mission

To restore hope, dignity, and opportunity for crisis-affected communities by delivering holistic humanitarian relief and long-term development programs.

Our vision

A world where every individual has access to basic needs, healthcare, education, and opportunities to rebuild their lives.


Where We Work


• Asia: Pakistan, Afghanistan, Bangladesh
• Middle East: Palestine, Syria, Jordan
• Africa: Sudan, Ethiopia, Somalia

Therapists without Borders partners, with local communities, governments, and global organizations to ensure culturally sensitive and impactful interventions.

CSS Files and Structure


Mainly three main CSS files are used in this theme. The first one bootstrap.css, second one is style.css which for this template and third one is responsive.css to control responsive layouts.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

Jquery and javascript


This theme imports three types of Javascript files.

  • jQuery
  • Custom scripts (theme.js)
  • Some plugins

  • jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  • Most of the animation in this site is carried out from the customs scripts. There are a few functions worth looking over.

In addition to the custom scripts, I have implemented few "tried and true" plugins to create the effects. This plugin is packed, so you won't need to manually edit anything in the file. The only necessary thing to know is how to call the method
